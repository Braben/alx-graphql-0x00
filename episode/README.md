# Character Queries

This directory contains GraphQL queries for fetching episodes by ID from the Rick and Morty GraphQL API.

## Endpoint

https://rickandmortyapi.com/graphql

## Queries Included

- character-page-1.graphql
- character-page-2.graphql
- character-page-3.graphql
- character-page-4.graphql

Each query retrieves:

-id,
-name,
-air_date,
-episode

The corresponding output files contain the API responses in JSON format.
