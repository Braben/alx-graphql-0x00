# Character Queries

This directory contains GraphQL queries for fetching characters by ID from the Rick and Morty GraphQL API.

## Endpoint

https://rickandmortyapi.com/graphql

## Queries Included

- character-id-1.graphql
- character-id-2.graphql
- character-id-3.graphql
- character-id-4.graphql

Each query retrieves:

- id
- name
- status
- species
- type
- gender

The corresponding output files contain the API responses in JSON format.
